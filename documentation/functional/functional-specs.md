# Functionele Specificaties - Pokemon Collectie App

- Overview van sets
- Overview van kaarten binnen een set
- Zoeken op een Set
- Zoeken naar een kaart
- Filteren van sets
- Filteren van kaarten op basis van bijvoorbeeld Naam, Rarity, Type, Subtype (Afhankelijk van mogelijkheden API)
- Per set in kunnen zien welke kaarten er in bezit zijn
- Per set in kunnen zien welke kaarten er ontbreken in eigen collectie
- In overzicht van sets kunnen zien welke er compleet zijn. (Wellicht completion percentage)
- Zelf custom een set aan kunnen maken
- Account aangemaakt worden om persoonlijke progress te tracken
- Offline toegang
- Mobiel toegankelijk
- Wishlist-functionaliteit op kaartniveau
- Binnen een set ook completeness percentage in kunnen zien
- Ik wil mijn collectie kunnen delen met anderen
- Zelfgemaakte deck kunnen delen met anderen
